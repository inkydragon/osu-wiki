# 단축키

이러한 단축키는 다양한 곳에 걸쳐서 사용할 수 있습니다:

## **게임 내에서**

-   **F7**: FPS 제한 모드를 변경
-   **F8**: 채팅 표시/숨기기
-   **F9**: 확장 채팅 표시 / 숨기기
-   **F12**: 화면을 캡쳐 (화면 공유를 위해 SHIFT 를 누르고 F12를 누르면 서버에 즉시 업로드 할수 있다.)

## **메뉴 키**

-   **F1**: 이전 곡
-   **F2**: 처음부터 재생
-   **F3**: 일시 정지 / 일시 정지 해제
-   **F4**: 정지
-   **F5**: 다음 곡
-   **F6**: 이동 (jukebox)
-   **Alt-Enter**: 전체 화면 모드로 전환
-   **Escape**: 하나의 메뉴로 돌아가기

## **메인 메뉴**

-   **P**: 플레이 모드 (osu!를 하려면 2번 눌러서 곡을 선택)
-   **P-S-T**: 플레이 모드 (Taiko 곡선택)
-   **P-S-C**: 플레이 모드 (Catch The Beat 곡선택)
-   **E**: 에디터 모드 (곡 선택)
-   **J**: 점프 (jukebox)
-   **Z/X/C/V/B**: Jukebox 컨트롤 (윈앰프 모양)
-   **D**: osu!direct (오직 서포터만 가능)
-   **O**: 옵션
-   **Q**: 나가기
-   **R**: 랜덤 곡

## **곡 선택**

-   **F1**: [모드 선택](/wiki/Game_Modifiers)
-   **F2**: 무작위의 맵 (즉시 랜덤곡을 선택하려면 빠르게 2번 누르십시오)
-   **F3**: Beatmap 옵션
-   **F5**: 새로 고침 목록 (복구 모드)
-   **Shift-Delete**: 현재 맵 삭제
-   **Ctrl-Enter**: Auto 모드로 현재 곡을 재생
-   **Ctrl-A**: Auto 모드로 전환
-   **Up/Down**: 선택 변경
-   **PageUp/PageDown**: 변경 선택 (대형으로)
-   **Enter**: 선택

## **플레이 모드**

-   **+/-**: 곡 오프셋 조정 (정밀도를 변경하려면 SHIFT / CTRL를 누른 상태)
-   **Escape**: 일시 정지 (Auto 모드를 사용하는 경우 곡을 종료)
